## 重要约定

> ⚠️ 在整个项目开发期间，必须遵守以下约定：

- 用户**不懂编程技术**，是初次使用 Claude Code

### 方案选择机制
当涉及技术选型、架构设计、实现方式等决策时，**必须**按以下流程：
1. 列出 2-4 个可行方案
2. 用**通俗语言**解释每个方案的优劣势
3. 给出**推荐建议**并说明理由
4. 由用户做出最终选择
5. 将决定记录回本文档

### 沟通要求
- 使用**中文**沟通
- 避免过于技术化的术语，使用时必须加以解释
- 每个阶段完成后，告知用户当前进度和下一步计划

### 记录要求
- 所有重要技术决策记录在本文档的「决策记录」部分

## 已知问题与注意事项

1. **ELECTRON_RUN_AS_NODE**：电脑系统环境变量中设置了此项，会导致 Electron 无法启动。已通过启动脚本自动移除。
2. **手机端首次加载慢**：需要下载约 600KB 的数据库引擎（WASM 文件），首次打开请等待 3-5 秒。
3. **桌面端与手机端数据不互通**：桌面数据在电脑本地文件，手机数据在浏览器 IndexedDB 中，两者独立。
4. **隧道链接临时性**：`serveo.net` 生成的公网链接在电脑重启后失效，需重新生成。
5. **清除手机浏览器数据会导致记账数据丢失**：建议定期在「设置」中备份。
